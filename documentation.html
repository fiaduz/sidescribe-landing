Documentation

Welcome to the SideScribe documentation. SideScribe is a browser extension designed to help researchers, students, and writers capture and organize information without leaving their browser tab.

Installation

Currently, SideScribe is available for Chrome and Brave browsers. To install the extension:

1. Visit the Chrome Web Store
2. Search for "SideScribe"
3. Click "Add to Chrome"

Quick Start

Once installed, look for the SideScribe icon in your extension toolbar. Pin it for easy access.

Tip: You can open the SideScribe sidebar anytime by pressing Alt + S.

Smart Highlighting

Simply select any text on a webpage. A small floating SideScribe button will appear. Click it to instantly save the highlight to your active project.
